## function:
```
move_dobot_gripper: input gripper and arduino object, output a moving motion for the dobot gripper
move_dobot_with_angle: input array containing q1, q2 and q3, output a moving motion for the entire dobot
move_dobot_with_pos: input array containing x, y and z, output a moving motion for the entire dobot
read_dobot_anlges: input arduino object, output q1, q2, and q3
read_dobot_pos: input arduino object, output x, y, z position of the end effector
```
